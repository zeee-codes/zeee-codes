### 👨‍💻 The Philosophy

I am not boxed into a specific niche. I operate as a **Software Generalist**, which means I believe writing code is only a fraction of the job. The real magic happens in the planning phase: breaking down complex problems, architecting clean and scalable systems, and figuring out the logic before a single line is written. 

- 🏢 Currently gaining industry experience as an Intern at **Kaevron Technologies**.
- ⚙️ **The Split:** 60% Problem Solver & System Designer / 40% Coder.
- 🎨 **Design Language:** Deeply inspired by tools like Notion and Linear. I thrive on creating dark mode interfaces, high-contrast visuals, smooth animations, and zero-clutter UI/UX.
